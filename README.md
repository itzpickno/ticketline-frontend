# TicketLine Frontend

Frontend do projeto final de Programação Web, desenvolvido com Next.js e React Bootstrap.

## Tecnologias

- Next.js
- React
- React Bootstrap
- Bootstrap
- Strapi REST API

## Funcionalidades

- Listagem de eventos
- Página de detalhe de evento
- Visualização de secções e bilhetes
- Login de utilizadores
- Registo de utilizadores
- Área do utilizador
- Painel de administração protegido por role
- CRUD de eventos
- CRUD de secções
- CRUD de bilhetes
- CRUD de categorias

## Requisitos cumpridos

- Frontend em Next.js
- Interface com React Bootstrap
- Backend em Strapi
- Consumo das APIs REST do Strapi
- Login de utilizadores
- Dois níveis de acesso:
    - Utilizador normal
    - Gestor
- CRUD sobre as tabelas principais
- Relação 1-N
- Relação M-N
- Swagger/OpenAPI no backend
